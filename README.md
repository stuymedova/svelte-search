# Svelte Search
Svelte app with a simple Search Component that filters an array of strings.

## Overview

```
.
└── src
    ├── components
    │   └── Search.svelte
    └── App.svelte
```
## Setup

Run in terminal:
```
npm i
npm run dev/build
```
